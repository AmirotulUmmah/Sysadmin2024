# Docker 101 Playground

Docker 101 adalah pengenalan dasar mengenai konsep dan penggunaan Docker, sebuah platform open-source untuk membangun, mengirimkan, dan menjalankan aplikasi dalam lingkungan terkontrol yang disebut container. Berikut adalah percobaan yang dilakukan mengikuti tutorial pada web berikut https://www.docker.com/101-tutorial/
